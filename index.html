Rectangular Art Planner

by Brian Gershon
briangershon.com GitHub
CascadiaJS AI Hackathon 2025
This project explores generative art through AI-assisted rectangle packing algorithms. By defining minimal primitives and leveraging language models for creative direction, it demonstrates how natural language can guide algorithmic art creation. Built using OpenAI models, Codex, and Claude Code, the approach combines mathematical constraints with artistic interpretation to generate unique visual compositions from simple textual descriptions.
