    Rectangle Joy

    Calming abstract scenes composed by an LLM
    Type a prompt and watch an LLM provided by OpenAI conduct thousands of rectangles into a meditative composition. The project blends lightweight math, WebGL shaders, and an Express bridge backed by Neon serverless Postgres to store each generated canvas for later replay.
